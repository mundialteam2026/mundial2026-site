# 🏆 Mundial 2026 - Stats en temps réel

Site web de statistiques officielles pour la Coupe du Monde 2026.

## Fonctionnalités
- Stats joueurs en temps réel (buts, passes, cartons)
- Scores live
- Classements
- Comparateur de joueurs

## Stack technique
- Frontend : Next.js 14 + TypeScript
- Backend : Node.js + Express
- BDD : PostgreSQL + Redis
